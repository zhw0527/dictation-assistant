# 听写助理

一个帮助学生进行听写练习的微信小程序

## 功能特点

- 📝 **智能听写**: 语音播报词汇,学生输入答案
- 📚 **历史记录**: 保存每次听写的成绩和详情
- 📥 **词汇导入**: 支持批量导入听写词汇
- 📊 **成绩统计**: 实时显示正确率和进度

## 项目结构

```
听写助理/
├── pages/              # 页面文件
│   ├── index/         # 首页
│   ├── dictation/     # 听写页面
│   └── history/       # 历史记录页面
├── utils/             # 工具函数
├── app.js             # 小程序入口
├── app.json           # 小程序配置
└── app.wxss           # 全局样式
```

## 开发指南

### 环境要求
- 微信开发者工具
- Node.js (可选)

### 开始开发

1. 使用微信开发者工具打开项目
2. 填写 AppID
3. 开始开发

### 使用 Cursor 开发

1. 用 Cursor 打开项目文件夹
2. 编辑代码文件
3. 在微信开发者工具中实时预览
